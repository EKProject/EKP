¿Qué necesitamos exactamente?

Necesitamos crear un software a modo de 'videojuego', donde podamos mover un personaje 3D haciendo uso de sensores de realidad virtual

Aquí puedes encontrar un ejemplo de software parecido a lo que nosotros buscamos, pese a que este sistema no cumple nuestras expectativas

Requisitos del programa

Cargar diferentes escenarios

Libre albedrío del personaje, en medida de lo posible

Interactuar con objetos diseñados para tal fin

Interfaz donde realizar los cambios al modelo de forma sencilla e intuitiva

Debe haber una cámara que puedas mover para mostrar los distintos planos y secuencias

Que admita un número alto de polígonos, para hacer decorados y modelos bien trabajados y definidos
